Requirement Requirements
------------------------

1.  Requirements **shall** be in `README.md`, Section `Requirements` or
    in a standalone markdown file named `REQUIREMENTS.md`.
2.  Requirements **shall** be written in
    [CommonMark](https://github.com/commonmark/CommonMark).
3.  Requirements **shall** have static analysis checks.
4.  Requirements **shall** use `LF` (`\n`) line endings.
5.  Requirements file **shall** be UTF-8 encoded.
6.  Requirements **shall** be written in active voice.
